Compute distance matrix based on an intersection over union metric.

Matrix are compared in order, with matrix i compared with matrix i+1

Parameters

M_s list of ndarrays The thresholded A matrices (masks) to compare, output of threshold_components

cm_s list of list of 2-ples the centroids of the components in each M_s

max_dist float maximum distance among centroids allowed between components. This corresponds to a distance at which two components are surely disjoined

Returns
D_s list of matrix distances
